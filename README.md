gowhois
===

gowhois is whois command implemented by Golang

## Installation
```sh
go install github.com/famasoon/gowhois@latest
```

## Usage
```sh
gowhois example.com
```

## Importing
```go
import (
    "github.com/famasoon/gowhois/whois"
)
```

## Example
### Get whois on example.com
```go
package main

import (
    "fmt"

    "github.com/famasoon/gowhois/whois"
)

func main() {
    result, _ := whois.Whois("example.com")
    fmt.Println(result)
}
```
